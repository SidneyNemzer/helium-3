# Helium 3

A turn-based robot battle/resource game. WIP.

See the [docs](./docs) folder for more info about the game.

![Game Screenshot](./docs/images/game.png)

## Developing

```
$ npm i
$ npm run start:server

# In another terminal
$ npm start
```

## Creating Server

```
$ terraform init
$ terraform apply
```

## Deploying

```
$ npm run deploy
```
